# Book Management Django Project
## Features

- Add a new book
- Update book details
- Delete a book
- View book details
- Search for books by title, author, or ID

## Prerequisites

- **Python**: 3.11.5
- **Django**: 5.1.3
## Installation
1. **Clone the repository**
   ```bash
   git clone https://github.com/alinsbtc/CRUD_BOOK.git
   cd book-management
